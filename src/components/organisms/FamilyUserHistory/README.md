# FamilyUserHistory

An organism component that displays routine history for family members.

## Usage

```vue
<FamilyUserHistory />
```

## Props

This component uses GraphQL queries to fetch family user data and doesn't accept props.

## Features

- Displays routine history for multiple family members
- Similar calendar view to UserHistory
- Task completion tracking per family member
- Interactive calendar navigation

## Dependencies

- GraphQL queries for family routine data
- Extends UserHistory functionality for multi-user scenarios
